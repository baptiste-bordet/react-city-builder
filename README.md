## Installation

Se positionner dans le dossier du projet (/manue)

### `npm i`
### `cd app`

## Lancement en local

Se positionner dans le dossier app (/manue/app)

### `npm start`

Le projet est basé sur [Create React App](https://github.com/facebookincubator/create-react-app).
